# About Energyze

Inspired by platforms like Fragrantica and Letterboxd, Energyze is a web application that allows users to browse a collection of drinks, 
view detailed information, rate products, save favorites, and search for new drinks based on their preferences.


### Features:
- browse a database of energy drinks
- search and filter by brand and flavor
- rate energy drinks
- vuew detailed information about each drink
- track popular porducts through view counts
- add/remove from favorite
- view your recent activity via recently viewed section
- light and dark mode support

### Technologies:
- ASP.NET Core
- Entity Framework Core
- SQLite
- HTML, CSS and JavaScript
- Blazor
- Bootstrap 5

### Running the Project
```
dotnet restore
dotnet ef database update
dotnet watch run
```
**The application will create and seed the database automatically on startup.**
